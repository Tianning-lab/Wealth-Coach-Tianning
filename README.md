## Production Version

The production wealth advisory chatbot is live at **[New Way Capital Advisory](https://nwc-advisory.com)**, powered by 17 financial APIs.

### Platform

- [nwc-advisory.com](https://nwc-advisory.com) — Financial analytics platform
- [Portfolio X-Ray](https://nwc-advisory.com/xray) — Free fund look-through analysis
- [Portfolio Consolidation](https://nwc-advisory.com/portfolio/consolidation) — Multi-bank portfolio merge
- [Property Comps](https://property.nwc-advisory.com) — 11-market comparable sales
- [API Documentation](https://api.nwc-advisory.com/docs) — Swagger UI for all endpoints
